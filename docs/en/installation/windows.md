# Windows

## Requirements

- `Python` = 3.10
- `Git`

---
## Installation

1. Clone Radiata repository

```bash
git clone https://github.com/ddPn08/Radiata.git
```

2. Enter the repository directory.

```bash
cd Radiata
```

3. Return to the root directory and run the application

```bash
.\launch-user.bat
```
