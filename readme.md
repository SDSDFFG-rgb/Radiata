# Installation

## Requirements

- Git
- python (3.10)
  - \*Older and newer may work but are not supported

## Windows

```
git clone https://github.com/ddPn08/Radiata.git
cd Radiata
./launch-user.bat
```

## Linux

```
git clone https://github.com/ddPn08/Radiata.git
cd Radiata
./launch.sh
```
